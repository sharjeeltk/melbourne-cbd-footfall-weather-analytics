# Melbourne CBD Footfall \& Weather Analytics

A portfolio data analytics project exploring pedestrian activity across Melbourne CBD and its relationship with weather conditions.





\## Business Context

A Melbourne retail or precinct-management organisation needs evidence to improve staffing, operating-hour, location and wet-weather planning decisions using pedestrian traffic and weather information.



\## Project Objective



\## Stakeholders

1. Retail Operations Manager

2\. Workforce Planning Manager

3\. Precinct or Location Manager



\## Business Questions

1. Which Melbourne CBD sensor locations have the highest pedestrian footfall?
2. Which hours and days experience the highest pedestrian activity?
3. How do weekday and weekend pedestrian patterns differ?
4. How do rainfall and temperature relate to pedestrian activity?
5. Which sensors or time periods contain missing, zero or potentially unreliable readings?
